# Virtual IT Infrastructure with Monitoring

## Overview
A fully virtualized IT infrastructure project simulating a real-world business network. Includes configuration of servers, firewalls, and monitoring systems to develop foundational system and network administration skills.

## Skills & Technologies
- **Windows Server 2022**: Active Directory, DNS, DHCP
- **Linux (Ubuntu/CentOS)**: SSH, users, cron, firewall
- **pfSense**: Firewall, NAT, VLANs
- **Nagios/Zabbix**: Network & host monitoring
- **VirtualBox / Proxmox**: VM orchestration and networking
